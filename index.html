async function fetchTokenBalance() { const tokenAddress = "0x3d6C000465a753BBf301b8E8F9f0c2a56BEC5e9b"; // Адрес твоего контракта const tokenABI = [ // ABI для твоего контракта, добавь сюда свой ABI ]; try { const accounts = await web3.eth.getAccounts(); const tokenContract = new web3.eth.Contract(tokenABI, tokenAddress); // Получаем баланс const balance = await tokenContract.methods.balanceOf(accounts[0]).call(); // Получаем количество десятичных знаков const decimals = await tokenContract.methods.decimals().call(); // Преобразуем баланс в нужный формат с учетом десятичных знаков const balanceInTokenUnits = balance / Math.pow(10, decimals); // Разделим на 10^decimals // Отображаем результат на странице document.getElementById('balance').textContent = `${balanceInTokenUnits} USDT`; } catch (error) { console.error("Ошибка при получении баланса токенов:", error); document.getElementById('balance').textContent = "Ошибка при получении баланса"; } }
